Невалидные токены:

Ключ: invalid_tokens
Значение: Множество (Set) невалидных токенов
Установка токена: SADD invalid_tokens <токен>
Проверка наличия токена: SISMEMBER invalid_tokens <токен>
Удаление токена: SREM invalid_tokens <токен>

Рефреш-токены:

Ключ: refresh_tokens
Значение: Множество (Set) рефреш-токенов
Установка токена: SADD refresh_tokens <рефреш-токен>
Проверка наличия токена: SISMEMBER refresh_tokens <рефреш-токен>
Удаление токена: SREM refresh_tokens <рефреш-токен>
